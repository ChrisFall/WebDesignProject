Myers Residence Virtual Tour Download Instructions


Instructions for downloading and running the 3D virtual tour application

To run this program, you must have a computer running Windows version 7 or higher and a mouse.

To download and install:

  1. Click the appropriate link below to download the file:
    • Download 32-bit Version Here
    • Download 64-bit Version Here
  2. Unzip the file. This may take several minutes.
  3. Go into the MyersExe folder and double click on the application MyersResidence.
  4. You may be prompted to install "DirectX" in order to run this program. This is a free download and is safe to install. This may also prompt a warning from your computer's firewall and ask you to allow access in order to execute.

Controls For First Person

  • W: move forward
  • A: move left
  • S: move backward
  • D: move right
  • Mouse Movement: turn/look
  • Esc or Q: Pause Menu (You can quit the game from the pause menu)

Instructions for downloading and running the Oculus Rift virtual tour application

Note: Running this virtual reality software requires Oculus Rift + Touch. If you have not already installed the Oculus Application software with your headset, we recommend that you do so and configure your sensors prior to running this application. We also recommend that you have a computer with the following specifications to run this software:

  • Graphics Card: NVIDIA GTX 1050Ti / AMD Radeon RX470 or greater
  • Alternative Graphics Card: NVIDIA GTX960 / AMD Radeon R9 290 or greater
  • CPU: Intel i3-6100 / AMD Ryzen 3 1200, FX4350 or greater
  • Memory: 8GB+ RAM
  • Video Output: Compatible HDMI 1.3 video output
